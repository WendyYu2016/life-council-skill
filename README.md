# life-council · 人生顾问委员会

> 「你缺的不是答案，是几双不同的眼睛同时看你的处境。」

一个 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) / Agent Skills 协议的 skill：把一个**真实的困惑**，交给几位你敬重的思想者（导师）**会诊**。

不是辩论抽象命题，也不是单一人物扮演——而是 **N 位导师各自用自己的镜片，照同一个人的真实处境**：帮 ta 看见自己没意识到的模式，再合成一条可以走的路。最后可铸成一张洞见卡片。

## 它解决什么

人生阶段的迷茫、重大决策的两难、意义感的缺失、转型期的心力问题、「我是谁 / 我要什么」这类**没有标准答案**的困惑。

## 与相邻 skill 的区别

| 需求 | 用哪个 |
|------|--------|
| 多视角**辩一个命题**（自由意志是否存在） | `ljg-roundtable` |
| 用**单一人物**视角看问题（芒格会怎么看） | 女娲的 `*-perspective` |
| **一个真人**的真实困惑，被几位导师**会诊 + 陪伴 + 合成行动** | **本 skill** |

## 工作流程

1. **倾听与澄清** — 先把人和困惑看清，再开会
2. **组委员会** — 选 3-4 位张力互补的导师（至少一位「意义型」+ 一位「决策型」）
3. **会诊** — 每位导师第一人称照一遍 → 主持人「照见模式 + 画结构图 + 合成可走的一步」
4. **铸卡**（可选） — 把最戳人的洞见 + 行动铸成一张 PNG 卡片

## 安装

```bash
# 方式一：克隆到 Claude Code 的 skills 目录
git clone https://github.com/barliwendy/life-council-skill.git ~/.claude/skills/life-council

# 方式二：手动把 SKILL.md + references/ 放进 ~/.claude/skills/life-council/
```

重启 Claude Code 后，说「顾问委员会」「帮我会诊一下」「我很迷茫」即可触发。

## 依赖（可选增强）

本 skill 可独立运行（导师视角即时生成），但配合以下 skill 效果更佳：

- **[女娲 nuwa-skill](https://github.com/alchaincyf/nuwa-skill)** — 提供已蒸馏的人物思维操作系统（芒格、乔布斯、费曼、Naval 等）。装了之后，委员会会直接读取 `~/.claude/skills/huashu-nuwa/examples/*/SKILL.md` 作为导师镜片。未安装时，skill 会即时生成导师视角。
- **[ljg-card](https://github.com/lijigang/ljg-skills)** — 把会诊洞见铸成 PNG 卡片（Phase 4）。需 Playwright。

> `references/mentor-pool.md` 内置了 4 位「意义型 / 哲学型」导师（稻盛和夫、维克多·弗兰克尔、马可·奥勒留、荣格）的现成镜片——这些是商业导师的盲区，专治意义感类困惑，无需额外蒸馏即可使用。

## 设计原则

- **以人为中心**：始终记得中央坐着一个真实的人，不是一道题。求真，但温暖。
- **看见 > 说教**：最高价值是让用户「啊，原来我是这样」。
- **落地**：再深的洞见，最后都收成一个明天就能走的下一步。
- **不灌鸡汤**：拒绝「你要相信自己」这类空话。
- **隐私分寸**：替第三方会诊时，只用当事人**主动分享过**的困惑，不带入打探来的或评价性的私下观察。
